# Esp32MotorCurrentGuard

`Esp32MotorCurrentGuard` 是电机电流保护公共库骨架。首版目标是支持 INA240A2 这类模拟电流检测链路，并把采样、滤波、阈值确认和诊断数据提供给应用层。

## 边界

- 本库不直接控制电机停机。
- 本库不保存配置。
- 本库不生成 Web 页面或图表。
- 应用层负责 ADC 采样、参数保存、告警展示和最终停机策略。

## 首版能力

- 启动宽限期。
- IIR 滤波。
- 预警阈值和故障阈值。
- 连续样本数与持续时间确认。
- 传感器异常按配置降级为忽略、预警或故障。
- `snapshot()` 和 `latestTracePoint()` 支持远程诊断与图表数据。

## INA240A2

`Ina240A2AnalogConfig` 记录首版 INA240A2 相关参数，例如 ADC 引脚、ADC 量程、放大倍数、采样电阻和零点偏移。它只描述换算参数；实际 ESP32 ADC 读取和校准由应用或适配层完成。
