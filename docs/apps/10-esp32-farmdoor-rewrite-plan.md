# Esp32FarmDoor 自动门重写方案

## 项目目标

Esp32FarmDoor 使用 Esp32Base 基础库重写鸡舍/养殖场自动门控制器，实现本地可靠开关门、状态查看、配置保存和故障保护。

首版只做本地设备控制和可靠运行，不接入 Blinker、MQTT 或云端控制协议。

## 硬件记录

- 主控：ESP32。
- 电机驱动：AT8236。
- 持久化存储：AT24C128。
- 电流检测：INA240A2。
- 物理限位：建议新增上限位/下限位开关。
- 电机：JGB37-545B 编码器直流减速电机。
- 电机供电：DC 12V。
- 当前默认减速比：131:1。

默认引脚：

- INA240A2 输出 ADC：GPIO33。
- 按钮 1：GPIO36。
- 开门按钮：GPIO39。
- 关门按钮：GPIO34。
- 停止按钮：GPIO35。
- 编码器 A 相：GPIO25。
- 编码器 B 相：GPIO26。
- AT8236 IN1/PWM1：GPIO16。
- AT8236 IN2/PWM2：GPIO17。
- I2C SDA：GPIO21。
- I2C SCL：GPIO22。

建议新增引脚：

- 上限位开关：待 PCB/GPIO 资源确认。
- 下限位开关：待 PCB/GPIO 资源确认。

上/下限位开关建议作为新硬件版本的 P0 设计项。无人值守运行时，单纯依赖编码器位置和电流保护仍可能在断电、编码器异常、配置错误、机械滑移后产生位置不可信问题；物理限位能提供独立的到位确认、远程归零和故障诊断依据。

电机接线：

- 红：电机电源 +。
- 黑：电机电源 -。
- 绿：Hall Sensor GND。
- 蓝：Hall Sensor Vcc。
- 黄：Hall Sensor A Vout。
- 白：Hall Sensor B Vout。

## 电机参数

- 额定电压：DC 12V。
- 默认减速比：131:1。
- 空载电流：不高于 200mA。
- 空载转速：47rpm。
- 额定转矩：10.5kg.cm。
- 额定转速：40rpm。
- 额定电流：不高于 1A。
- 堵转电流：5A。
- 编码器输出：AB 相方波，相位差约 90 度。
- 编码器脉冲：电机轴每圈 A/B 各 16 个脉冲。
- 默认输出轴每圈信号数：16 * 131 = 2096。

减速比、电机轴每圈脉冲数、输出轴每圈信号数必须作为配置参数，便于后续更换电机。

## 功能范围

首版功能：

- 本地 Web 控制：开门、关门、停止、状态查看、配置保存。
- 物理按钮：开门、关门、停止。
- 编码器位置：累计信号数用于判断开门、关门到位。
- 物理上/下限位：用于到位确认、归零、异常方向检测和远程恢复。
- 归零：优先通过远程维护命令低速运行到下限位完成，不依赖现场人工确认。
- 电流保护：通过 INA240A2 检测过流和辅助判断堵转。
- 运行保护：编码器无脉冲堵转、最大运行时间、最大运行脉冲。
- 持久化：使用 AT24C128 新格式保存配置、校准参数、当前位置和关键状态。
- 日志：记录启动、开门、关门、停止、归零、故障等关键事件。

明确不做：

- 不接入 Blinker App。
- 不实现 MQTT 或 HTTP 云端控制。
- 不迁移老项目 AT24C128 数据。
- 不修改 Esp32Base。
- 不修改 old_prj。
- 当前阶段不写源码、不创建 PlatformIO 工程结构。

## 控制模型

Esp32FarmDoor 应由应用层 `DoorController` 组合公共库：

```text
DoorController
  Esp32EncodedDcMotor
  Esp32MotorCurrentGuard
  Esp32At24cRecordStore
```

开门是运行到配置的开门目标位置。关门是运行回零或运行到配置的关门位置。停止命令应优先保证安全，可根据配置选择急停、刹车或软停止后刹车。

编码器仍是主要位置控制依据；物理限位作为独立安全输入和位置校准依据，不替代编码器闭环。推荐策略：

- 开门过程中触发上限位：立即按到位策略停机，确认当前位置为开门目标位置，保存状态。
- 关门或归零过程中触发下限位：立即按到位策略停机，确认当前位置为 0，保存状态。
- 运动方向与触发限位不一致：立即故障停机，例如开门时触发下限位。
- 上下限位同时有效：判定为传感器、接线或机械异常，不允许继续运动。
- 到达目标编码器脉冲但未触发预期限位：允许按配置决定是否视为到位、警告或故障；首版建议自动门使用限位时将其视为故障，避免远程误判。

限位开关建议采用常闭 NC 接法，并在软件中支持极性配置。常闭接法可在断线时更容易暴露为故障。若预算和结构允许，优先选择带强制断开触点的工业限位开关。输入需要做防抖和异常检测，默认稳定 50ms 后才采信；硬件上应考虑外部上拉/下拉、RC 滤波、线缆固定和防水防尘。

断电重启恢复策略：

- 下限位有效：位置可信，进入已关闭状态，当前位置置 0。
- 上限位有效：位置可信，进入已打开状态，当前位置置为开门目标。
- 上下限位同时有效：进入故障状态，需要远程诊断；若无法排除，提示现场处理。
- 两个限位均无效：优先读取持久化位置作为估计值，但应标记为非完全可信；允许远程发起低速归零到下限位。

现场人工操作不应作为正常流程的一部分。只有限位冲突、常闭回路断线、反复归零失败、机械卡死或远程观察无法确认风险时，才提示需要现场处理。

## 电流保护方案

INA240A2 按双向测量设计：

- INA240A2 增益：50V/V。
- 默认 Rsense：5mΩ，最终值需要实机确认。
- 电流保护默认开启。
- 电流保护可以关闭。
- 默认电流阈值先按 2500mA 设计，实机测试后调整。

保护判定应包含：

- 启动宽限，避免启动瞬间误判。
- 滤波，避免 ADC 抖动误判。
- 连续超限确认。
- 传感器异常检测。
- 故障停机和事件记录。

## 持久化数据

需要保存：

- 电机速度。
- 软启动时长。
- 软停止时长。
- 开门目标圈数或目标脉冲。
- 微调圈数。
- 减速比。
- 编码器电机轴每圈脉冲数。
- 输出轴每圈信号数。
- 当前位置编码器信号数。
- 电流保护开关。
- 电流阈值。
- Rsense 采样电阻值。
- INA240A2 零点校准参数。
- 关键校准参数。

配置变更时保存。当前位置应在电机停止、故障停机、归零后保存，不应高频持续写入。
