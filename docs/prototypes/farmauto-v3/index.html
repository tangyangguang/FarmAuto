系统设置

集中管理会影响长期自动运行的参数；日常计划页只设置每天几点执行什么。

参数正常

RS485 通讯参数

通讯方式RS485 主控轮询总线
作用：主控与门控、下料分站通讯。可取值：第一版固定，不提供其他连接方式。
设备地址范围1-15
作用：扫描和绑定分站拨码地址。可取值：1-15 可绑定；0 是保留地址，不允许普通设备使用。
通讯速率待工程固化
作用：主控和分站串口通讯速率。可取值：必须与分站固件一致；当前项目文档未定具体波特率。

扫描与离线判定

扫描地址1-15
作用：设备管理里主动扫描哪些 RS485 地址在线。可取值：第一版固定 1-15；地址 0 只做异常提示。
正常轮询间隔待工程固化
作用：主控后台检查已启用设备是否在线。可取值：需结合总线数量和分站响应时间确认，不能在原型里伪造默认值。
离线判定待工程固化
作用：决定什么时候写入设备通讯记录并触发异常提醒。可取值：建议按“连续失败次数 + 最近在线时间”实现，具体次数由工程方案确定。

温湿度记录

首页读数刷新时读取最新值
作用：首页显示当前温度和湿度。可取值：固定为实时读取，不按小时缓存显示。
历史保存每小时 1 个点
作用：长期查看温湿度变化。可取值：第一版固定，不在 Web 页面修改。

自动运行规则

  • ✓暂停只暂停自动执行，暂停期间允许手动操作。
  • !暂停期间错过的计划默认不补跑。
  • ✓供电恢复报告自动生成，历史报告在系统事件里查看。
  • ✓门控到位、下料转动和异常停止由分站本地闭环完成。

这些是产品规则，不作为普通开关暴露；需要改规则时应先回到主方案裁决。
